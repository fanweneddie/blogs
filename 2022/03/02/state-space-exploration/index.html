Discussion

In two algorithms above, we use reachability algorithm that visits each state individually for assertion verification. So we call it explicit state verification.

If state spaces the infinite, this explicit exploration does not terminate since each newly-found state is new in RSRSRS or BRSBRSBRS. Therefore, in this case explicit state verification can only prove the presence of bugs rather than the absence of bugs.

If state spaces is finite, this exploration can terminate and find both the absence and presence of bugs.
