Note of Regular Language

Created2022-03-23Updated2022-03-23

This is my simple note of Regular Language, which references to Prof. Wenchao Huang’s slide. I only list the framework and leave out the relatively complex proof.

Concept

  1. A deterministic finite automaton (DFA) is a 5-tuple <Q,Σ,δ,q0,F><Q, \Sigma, \delta, q_0, F><Q,Σ,δ,q0​,F>, where δ:Q×Σ→Q\delta: Q \times \Sigma \rightarrow Qδ:Q×Σ→Q is the transition function; A non-deterministic finite automaton (NFA) is a 5-tuple <Q,Σ,δ,q0,F><Q, \Sigma, \delta, q_0, F><Q,Σ,δ,q0​,F>, where δ:Q×Σϵ→P(Q)\delta: Q \times \Sigma_{\epsilon} \rightarrow P(Q)δ:Q×Σϵ​→P(Q) is the transition function.

  2. A language is called regular language if some DFA recognizes it.

  3. Regular operations are union, concat and star over languages.

  4. Regular expressions are defined by regular operations. Formally, we say RRR is a regular expression if RRR is {a}∣{ϵ}∣∅∣(R1∪R2)∣(R1∘R2)∣(R1∗)\{ a \} \mid \{ \epsilon\} \mid \empty \mid (R_1 \cup R_2) \mid (R_1 \circ R_2) \mid (R_1^*){a}∣{ϵ}∣∅∣(R1​∪R2​)∣(R1​∘R2​)∣(R1∗​).

Theorem

  1. NFAs and DFAs are equivalent. (For each NFA, we can use subset construction to build its equivalent DFA.)

  2. A language is regular iff some NFA recognizes it.

  3. Regular languages are closed under regular operations. (We can construct the new DFA from NFA).

  4. If a language is described by a regular expression, then it is regular (Proof is similar to Theorem 3); If a language is regular, then it is described by a regular expression (We can convert DFA to GNFA, and then simplify that GNFA to a regular expression). Therefore, A language is a regular language iff it can be expressed by a regular expression.

  5. Pumping lemma shows the property of all regular languages: All strings that are at least as long as ppp in a regular language can be expressed as xyizxy^izxyiz, where i≥0i \geq 0i≥0 and ∣y∣>0|y| > 0∣y∣>0 and ∣xy∣≥p|xy| \geq p∣xy∣≥p. This lemma can be easily proved in the case that DFA contains a cycle, and it is used to prove that a language (such as {0n1n∣n≥0}\{ 0^n1^n \mid n \geq 0 \}{0n1n∣n≥0}) is not a regular language.
