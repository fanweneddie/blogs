Lattices

Partial Order

A partial order is a set SSS equipped with a binary relation ⊑\sqsubseteq⊑ where the following conditions are satisfied:

  • Reflexivity: ∀x∈S:x⊑x\forall x \in S: x \sqsubseteq x∀x∈S:x⊑x

  • Transitivity: ∀x,y,z∈S,x⊑y ∧y⊑z ⇒x⊑z\forall x,y,z \in S, x \sqsubseteq y \ \land y \sqsubseteq z \ \Rightarrow x\sqsubseteq z∀x,y,z∈S,x⊑y ∧y⊑z ⇒x⊑z

  • Anti-symmetry: ∀x,y∈S,x⊑y ∧y⊑x⇒x=y\forall x,y \in S, x \sqsubseteq y \ \land y \sqsubseteq x \Rightarrow x = y∀x,y∈S,x⊑y ∧y⊑x⇒x=y

Lattice

A lattice is a partial order (S,⊑)(S,\sqsubseteq)(S,⊑) in which x⊔yx \sqcup yx⊔y and x⊓yx \sqcap yx⊓y exist for all x,y∈Sx,y \in Sx,y∈S.

A complete lattice is a partial order (S,⊑)(S,\sqsubseteq)(S,⊑) in which ⊔X\sqcup X⊔X and ⊓X\sqcap X⊓X exist for all X⊆SX \subseteq SX⊆S.

Every lattice has a unique largest element denoted ⊤\top⊤ and a unique smallest element denoted ⊥\bot⊥.

The height of a lattice is defined to be the length of the longest path from ⊤\top⊤ to ⊥\bot⊥.

Fixed-point theorem

In a complete lattice LLL with finite height, every monotone function f:L→Lf : L \rightarrow Lf:L→L has a unique least fixed point denoted lfp(f)lfp(f)lfp(f) defined as

lfp(f)=∪i≥0fi(⊥)lfp(f) = \cup_{i \geq 0} f^i(\bot) lfp(f)=∪i≥0​fi(⊥)

proof:

Since ⊥\bot⊥ is the least element, ⊥⊑f(⊥)\bot \sqsubseteq f(\bot)⊥⊑f(⊥).

Since fff is monotone, f(⊥)⊑f(f(⊥))=f2(⊥)f(\bot) \sqsubseteq f(f(\bot)) = f^2(\bot)f(⊥)⊑f(f(⊥))=f2(⊥), and by induction we have fi(⊥)⊑f(fi(⊥))=fi+1(⊥)f^{i}(\bot) \sqsubseteq f(f^{i}(\bot)) = f^{i+1}(\bot)fi(⊥)⊑f(fi(⊥))=fi+1(⊥).

Therefore, we have an increasing chain:

⊥⊑f(⊥)⊑f2(⊥)⊑...\bot \sqsubseteq f(\bot) \sqsubseteq f^2(\bot) \sqsubseteq... ⊥⊑f(⊥)⊑f2(⊥)⊑...

Since LLL has a finite height, there must exist kkk such that fk(⊥)=fk+1(⊥)f^k(\bot) = f^{k+1}(\bot)fk(⊥)=fk+1(⊥). It is obvious that fk(⊥)f^k(\bot)fk(⊥) is a fixed point for fff and fk(⊥)f^k(\bot)fk(⊥) is the least upper bound for all the elements in the chain, so we first guess that lfp(f)=fk(⊥)lfp(f) = f^k(\bot)lfp(f)=fk(⊥).

Assume that xxx is another fixed point.

Since ⊥⊑x\bot \sqsubseteq x⊥⊑x and fff is monotone, we have f(⊥)⊑f(x)=xf(\bot) \sqsubseteq f(x) = xf(⊥)⊑f(x)=x. So we also have f2(⊥)=f(f(⊥))⊑f(f(x))=f(x)=xf^2(\bot) = f(f(\bot)) \sqsubseteq f(f(x)) = f(x) = xf2(⊥)=f(f(⊥))⊑f(f(x))=f(x)=x.

By induction, we have fk(⊥)⊑xf^k(\bot) \sqsubseteq xfk(⊥)⊑x. Therefore, fk(⊥)f^k(\bot)fk(⊥) is a least fixed point and the guess of lfp(f)=fk(⊥)lfp(f) = f^k(\bot)lfp(f)=fk(⊥) is correct.

If there are two least fixed point x1x_1x1​ and x2x_2x2​, then based on the result above, we have x1⊑x2x_1 \sqsubseteq x_2x1​⊑x2​ and x2⊑x1x_2 \sqsubseteq x_1x2​⊑x1​. By anti-symmetry, we can know that the least fixed point is unique.

This theorem tells us not only that equation systems over complete lattices always have solutions, provided that the lattices have finite height and the constraint functions are monotone, but also that uniquely most precise solutions always exist.
